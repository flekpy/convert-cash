### Convert Cash

##### структура

```
├── main.tsx       # Точка входа
└── src
    ├── components # UI компоненты
    ├── lib/api    # http запросы
    ├── store      # redux-хранилище
    └── hooks      # хуки
```

#### Для запуска

```bash
1. клонируем репозиторий

устанавливаем зависимости
2. yarn

запускаем на localhost:3000
3. yarn watch
```
